# Lotto Number Generator

## Overview

A simple, visually appealing web application to generate random lottery numbers. The application will feature a clean, modern design with a touch of animation to enhance the user experience.

## Project Outline

### Style and Design

*   **Layout:** A centered, single-column layout that is mobile-responsive.
*   **Color Palette:** A vibrant and energetic color scheme.
    *   Background: A subtle noise texture.
    *   Primary Colors: Bright and engaging colors for buttons and highlights.
    *   Text: Clear and readable typography with good contrast.
*   **Typography:** Expressive and modern fonts (e.g., from Google Fonts).
*   **Visual Effects:**
    *   **3D Effect:** Multi-layered drop shadows on containers to create a sense of depth.
    *   **Glow Effect:** A subtle glow on interactive elements like the button.
*   **Iconography:** Use of a simple, relevant icon for the lottery theme.
*   **Animation:**
    *   The generated numbers will animate into view.
    *   The "Generate" button will have a subtle animation on hover/click.

### Features

*   **Number Generation:** Generate 6 unique random numbers between 1 and 45.
*   **Number Display:** The generated numbers will be displayed in visually distinct elements (e.g., "lotto balls").
*   **"Generate" Button:** A clear call-to-action button to generate a new set of numbers.
